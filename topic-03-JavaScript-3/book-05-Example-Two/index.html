This lab will use JavaScript code to display and modify the contents of an array (in an unordered list).

Now, we will look code for the A Dream of the Future application.

  • Display the Array
  • Add a Dream

Display the Array

The script creates and populates an array of dreams/wishes:

const dreams = [
  'Find and count some sheep',
  'Climb a really tall snowy mountain',
  'Wash the dishes'
];

We use const here as the array cannot be reassigned to a new value.

The following code executes as it is not inside a function:

dreams.forEach( function(currentdream) {
  appendNewDream(currentdream);
});

The .forEach() method calls a provided function once for each element in an array, in order. As you can see, for each element we call the appendNewDream function and pass the current element (dream) value.

The appendNewDream function:

  • Creates an empty list item (<li></li>);
  • Sets the content/text of the new list item to the value of the passed in array element, and;
  • Adds the list item to the unordered list.
function appendNewDream(dream) {
  let newListItem = document.createElement('li');//create a new list item
  newListItem.innerHTML = dream; //set text of new list item to dream
  dreamsList.appendChild(newListItem); //add list item to the unordered list
}

Add a Dream

A form is displayed that includes a text box and Submit button.

      <form id ="adddream" class = "ui form">
          <div class="fields">
             <div class="eight wide field">
                   <input name="dream" type="text" maxlength="100" placeholder="Dreams!" required>
             </div>
             <div class="field">
           <button type="submit" id="submit-dream" class = "ui button">Submit Dream</button>
             </div>
           </div>
       </form>

Once the user enters some text and clicks the Submit button (Submit Dream) the following code is executed:

let dreamsForm = document.getElementById('adddream');

dreamsForm.addEventListener('submit', pushnewdream, false);

function pushnewdream(event) {
  // stop our form submission from refreshing the page
  event.preventDefault();

  // get dream value and add it to the list
  dreams.push(dreamInput.value);
  appendNewDream(dreamInput.value);

  // reset form
  dreamInput.value = '';
  dreamInput.focus();
};

This code ascertains the value of the text box (dreamInput.value) and pushes it onto the array (dreams.push(dreamInput.value);). The appendNewDream function is called again to display it as a list item at the end of the unordered list. The text box is cleared (dreamInput.value = '';) and the cursor is placed back in the text box (dreamInput.focus();).
