Add an Employee

When the user enters employee data (id, firstname, lastname, county, and email) and then clicks the submit (Add Employee) button the text box data is put together into a new object, and this new object is pushed onto the employee array. The page is then refreshed including the new employee object. The modal dialog is hidden and the text boxes are cleared.

$('#emplist').submit(function() {
  newemp= {
     id:$('#id').val(),
     firstname:$('#firstname').val(),
     lastname:$('#lastname').val(),
     county:$('#county').val(),
     email:$('#email').val()
  }

  employee.push(newemp);
  var theCompiledHtml = template(employee);
  $("#content-placeholder").html(theCompiledHtml);
  $('#modalForm').modal('hide');
  $('input[type="text"]').each(function() {
        $(this).val("");
  });
  $('input[type="email"]').val("");
  return false;
});
