Where to Place it

Since JavaScript isn't HTML, you will need to let the browser know in advance when you enter JavaScript to an HTML page. This is done using the <script> tag. The browser will use the <script> and </script> to tell where JavaScript starts and ends. In higher versions of HTML, such as HTML5 you can use <script> and </script> instead.

The word alert is a standard JavaScript command that will cause an alert box to pop up on the screen. The visitor will need to click the "OK" button in the alert box to proceed.

By entering the alert command between the <script> and </script> tags, the browser will recognize it as a JavaScript command. If we had not entered the <script> tags, the browser would simply recognize it as pure text, and just write it on the screen. You can enter JavaScript in both the <head> and <body> sections of the document. 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>Using alert()</title>
</head>
<body>
<script>
  alert("Welcome to my world!!!");
</script>
</body>
</html>

Facts to Note

Knowing that JavaScript needs to be entered between <script> tags, is a start. But there are a few other things you need to know before writing your first JavaScript:

-JavaScript lines end with a semicolon.
 You may have noticed from the example on the previous page that JavaScript lines end with a semicolon.
 You can easily put all your JavaScript on a single line without destroying the performance of it.However, you would destroy the overview of your script so it is not advisable.

-Always put string text within quotes (single or double).
 When entering text to be handled by JavaScript, you should always put the text within quotes.
 If you forget to enclose your text in " ", JavaScript will interpret your text as being variables rather than text.

-Capital letters are different from lowercase letters.
 You should always remember that capital letters are different from lowercase letters.
 This means that when you write commands in JavaScript, you need to type capital letters in the correct places, and nowhere else.

Document.Write

Consider this Example:

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<title>Using document.write()</title>
</head>
<body>
<script>
  document.write("Welcome to my world!!!");
</script>
</body>
</html>

Instead of having JavaScript write something in a popup box we could have it written directly into the document.

The document.write is a JavaScript command telling the browser that what follows within the parentheses is to be written into the document.

Note: When entering text in JavaScript you need to include it in " ".
