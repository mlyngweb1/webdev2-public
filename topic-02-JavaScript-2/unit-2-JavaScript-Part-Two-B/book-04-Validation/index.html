Objectives

We will now combine string functions and our own declared functions to perform validation on strings input by the user.

  • Empty Strings
  • Numeric Strings (Numbers Only)
  • Repeat until valid input entered

Empty Strings

We can check whether the input is empty or whether the user has selected cancel as in the following example: 

<script>
let sentence = prompt("Enter a string, please","");
let res = isFilled(sentence);
if (res) {
   document.write("<p>The string entered was " + sentence+ "</p>");
}
else document.write("<p>No input was entered</p>");

function isFilled(field) {
 if (field==null || field.length== 0) {
     return false;
  }
 return true;
}
</script>

Numeric Strings (Numbers Only)

We can check whether the string entered are has numbers only as in the following example: 

<script>
let mynum = parseInt(prompt("Enter your favourite number, please",""));
let res= isFilled(mynum);
if (res) {
   document.write("<p>The number entered was " + mynum+ "</p>");
}
else document.write("<p>The valid entered was invalid</p>");


function isFilled(field) {
 if (field==null || field.length == 0 ||  (isNaN(field))) {
     return false;
  }
 return true;
}
</script>

Repeat until valid input entered

The isFilled function is modified to include a While statement as in the following example:

let sentence = prompt("Enter a string, please","");
sentence = isFilled(sentence)
document.write("<p>The string entered was " + sentence+ "</p>");

 while (field==null || field.length== 0) {
