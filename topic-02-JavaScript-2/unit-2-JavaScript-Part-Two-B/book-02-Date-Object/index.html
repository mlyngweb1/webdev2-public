Objectives

This lab will look at the various Date methods.

JavaScript includes a well developed Date class which provides methods to perform many different date manipulations.

  • Using the Date object

  • Retrieving values using the Date object

  • Setting values using the Date object

Using the Date object

Date()
Constructs an empty date object.
For example: var d = new Date();
Date(milliseconds)
Constructs a new Date object based on the number of milliseconds that have elapsed since 00:00:00 hours on 01/01/1970.
For example: var d = new Date(9999999999);
Date(string)
Creates a Date object based upon the contents of a text string. The string must be in the format which is created by the Date.parse() function.
For example: var d = new Date("January 1, 2019 00:00:01");
Date(year, month, day[, hour, minute, second])
Creates a new Date object based upon the numerical values for the year, month and day. Optional time values may be supplied. January is represented by 0, and December by 11.
For example: var d = new Date(2018,11,25,11,33,30,0);

Retrieving values using the Date object

The following lists some of the more useful methods for retrieving values from the Date object.

getDate()
Returns the day of the month.
getDay()
Returns an integer representing the day of the week, Sunday is 0 and Saturday is 6.
getMonth()
Returns the month field of the Date object, represented by an integer, January is 0 and December is 11.
getFullYear()
Returns the year as a four digit number.
getHours()
Returns the hour field of the Date object.
getMinutes()
Returns the minutes field of the Date object, from 0 to 59.
getSeconds()
Returns the seconds field of the Date object.
getMilliseconds()
Returns the milliseconds field of the Date object as a number from 0 to 999.
getTime()
Returns the number of milliseconds since midnight on 01/01/1970 which the Date object represents.

Setting values using the Date object

The following lists some of the more useful methods for setting values from the Date object.

setDate(day)
Sets the day value of the object. Accepts values in the range 1 to 31.
setFullYear(year[, month, day])
Sets the year value of the object. Optionally also sets month and day values. All are passed as integers. Year as a four digit date, month in the range 0 to 11, day in the range 0 to 31.
setHours(hours[, mins, secs, ms])
Sets the hours value of the object to an integer in the range 0 through 23. Optionally also sets minutes, seconds and milliseconds.
setMinutes(min[, secs, ms])
Sets the minutes value using an integer in the range 0 to 59.
setSeconds(secs[, ms])
Sets the seconds value using an integer in the range 0 to 59.
setMilliseconds(ms)
Sets the milliseconds value of the object in the range 0 to 999.
setMonth(month[, day])
Sets the month value using an integer in the range 0 to 11.
setTime(time)
Sets the value of the Date object. The parameter is a string representing the number of milliseconds since midnight, 01/01/1970.
