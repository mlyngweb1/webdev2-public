Math Object Operations

All of the specialised mathematical operations are available through the Math Object. To use these simply put the name of the operation after the keyword Math. and include any parameters inside the parentheses.

abs(value)
returns the absolute value of the number passed into it.
ceil(value)
returns the smallest integer which is greater than, or equal to, the value passed in.
cos(value), sin(value), tan(value)
returns the cosine, sin and tangent, respectively, of the value passed in.
floor(value)
returns the largest integer which is smaller than, or equal to, the number passed in.
max(value1, value2, ..., valueN)
returns the larger of its arguments.
min(value1, value2, ..., valueN)
returns the smaller of its arguments.
pow(value, power)
returns the result of raising value to power.
round(value)
returns the result of rounding its argument to the nearest integer.
sqrt(value)
returns the square root of the value.

Generating Random Numbers

Math.random() returns a pseudorandom number between 0 and 1. The random number generator is seeded from the current time.The function always delivers a 16-place decimal that ranges from 0.0000000000000000 to 0.9999999999999999. If you want to get a random number between 1 and another number, just multiply the random() method by the uppermost number and add 1 to the total.

For Example: to generate a random number from 1 to 10:

let bigDecimal = Math.random();
let improvedNum = (bigDecimal * 10) + 1;
let mynumber = Math.floor(improvedNum);

This is what happens in the code above, line by line:

  • Line 1: Generates a 16-place decimal and assigns it to the variable bigDecimal.
  • Line 2: Converts the 16-place decimal to a number ranging from 0.0000000000000000 through 9.9999999999999999, then adds 1, so the range winds up 1.0000000000000000 through 10.9999999999999999. This number is assigned to the variable improvedNum.
  • Line 3: Rounds down the value represented by improvedNum down to the nearest integer that ranges from 1 through 10 and assigns it to mynumber.

A shorthand way of writing the above is:

let mynumber =Math.floor(Math.random()* 10 + 1);

If you want to get a random number between 2 numbers where the first number is not 1 then use the following calculation:

let mynumber =Math.floor(Math.random()*(max - min + 1) + min);

Formatting Currency Values

toFixed(), which lets you convert a number to a string that matches the number of decimal places you want.

let printCost = '&euro;'+ cost.toFixed(2);
